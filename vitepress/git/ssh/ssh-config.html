ssh 使用 Host 远程登录服务器 ​

sh
# 1. 在本机生成 ssh 公钥 和 私钥
ssh -t rsa/ed25519 -c "<email>"

# 2. 使用用户名密码先登录远程主机
ssh <username>@<ip/host:port>

# 3. 将生成的 公钥 写入远程主机的 /<username>/.ssh/authorized_keys 文件中

# 4. 在 本地的 ~/.ssh/config 文件中配置 Host
##
# Host <登录时使用的 host 名称>
#     HostName <ip/host>
#     User <username>
#     Port <port>
#     IdentityFile <生成的私钥目录>
##

# 4.1 也可以将私钥目录添加到 ssh 验证私钥目录中
ssh-add <私钥目录>

# 5. 登录远程主机
ssh <配置 Host 时填写的 登录时使用的 host 名称>
